## 🚀 Overview

This repository contains the source code for a fully responsive, static developer portfolio website. The project was built from scratch without relying on external UI frameworks to demonstrate a strong core understanding of foundational web technologies. 

The website features a modern design, smooth scrolling, a mobile-friendly navigation menu, and structured sections to display personal information, technical skills, and featured projects.

## 🛠️ Technologies Used

* **HTML5:** For semantic and accessible structural layout.
* **CSS3:** For styling, custom color variables, Flexbox/Grid layouts, and media queries for mobile responsiveness.
* **JavaScript (Vanilla):** For interactive UI elements, specifically the mobile hamburger menu toggle and dynamic navbar styling on scroll.
* **FontAwesome:** For lightweight, scalable vector icons.
